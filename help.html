STZ压缩解压工具 - 帮助文档

1. 功能介绍

STZ压缩解压工具是一款支持多种压缩算法的文件压缩和解压工具，具有直观的图形界面和丰富的功能。

2. 快速开始

2.1 压缩文件

  1. 点击"添加文件"或"添加文件夹"按钮，选择要压缩的文件或文件夹
  2. 选择压缩级别（1-9，数字越大压缩率越高但速度越慢）
  3. （可选）勾选"设置密码"并输入密码，为压缩包添加加密保护
  4. 点击"开始压缩"按钮，选择保存路径，开始压缩过程

2.2 解压文件

  1. 在"解压"标签页中，点击"选择文件"按钮，选择要解压的压缩包
  2. （可选）如果压缩包有密码，勾选"输入密码"并输入正确的密码
  3. 点击"开始解压"按钮，选择解压路径，开始解压过程

3. 打包指南

3.1 本地部署

  1. 确保已安装Python 3.7或更高版本
  2. 安装所需依赖：pip install -r requirements.txt
  3. 运行程序：python main.py

3.2 打包参数配置

支持的打包参数：

  • 压缩级别：1-9，控制压缩率和速度的平衡
  • 加密选项：是否为压缩包添加密码保护
  • 压缩算法：自动选择最佳算法（支持zlib、bz2、lzma等）
  • 输出格式：支持.stz、.zip、.tar.gz等多种格式

3.3 高级配置

在配置文件中可以设置以下高级参数：

{
    "default_compression_level": 6,
    "language": "zh",
    "theme": "system",
    "output_format": "stz",
    "auto_open_output": true
}

4. 常见问题解决方案

4.1 压缩失败

  • 检查文件路径是否包含特殊字符
  • 确保有足够的磁盘空间
  • 检查是否有权限访问要压缩的文件

4.2 解压失败

  • 检查压缩包是否损坏
  • 确保输入了正确的密码（如果有密码保护）
  • 检查是否有权限写入解压路径

4.3 程序崩溃

  • 确保已安装所有依赖
  • 尝试以管理员身份运行程序
  • 检查日志文件获取详细错误信息

5. 技术支持

如果您在使用过程中遇到问题，可以：

  • 查看日志文件获取详细错误信息
  • 检查配置文件是否正确
  • 尝试重新安装依赖
注意：为了保证最佳性能，建议定期清理临时文件和日志文件。
